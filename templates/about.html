Stock Sentiment Analysis & Prediction

In phase 1, building an application which allows user to build portfolio of personal favorite stocks and "powerful" people in finance. This application will track performance of the stocks in the portfolio and provide day to day sentiment on all stocks based on "powerful" people in finance selected.

In phase 2, plan is to predict stock performance based on data collected from phase 1.

For phase 1, stock data is extracted from yahoo finance and sentiment results are extracted from Twitter using Twitter API. For phase 2, machine learning related modules will be used (More to come soon!!!)
